# Hidden Lineage - Public Dashboard

Research-based ancestry analysis using ancient DNA databases and over 1.1 million SNPs.

🔗 **Live Dashboard**: https://nagyelshal.github.io/hidden-lineage-public/

## Overview

Hidden Lineage is a public experiment to uncover deep ancestral roots using research-based methods, going far beyond commercial testing by incorporating thousands of years of genetic history and transparent, open-source methodologies.

## Key Features

- **20,134 total genomes** analyzed
- **9,121 ancient DNA samples** (AADR database)
- **1.1M+ SNPs** processed
- **4,000+ years** of genetic history
- **Open-source methodologies** with statistical validation

## Analysis Status

Real-time analysis dashboard showing current progress of ADMIXTURE analysis and upcoming results visualization.

## Methodology

This project uses established population genetics methods and tools, including:
- ADMIXTURE analysis for ancestry composition
- Ancient DNA databases (AADR)
- Modern reference panels (1000 Genomes)
- Statistical validation with high confidence thresholds

## Transparency

This is an independent research project. While every effort was made to follow established scientific standards, it is not affiliated with any academic institution or published in peer-reviewed journals.
