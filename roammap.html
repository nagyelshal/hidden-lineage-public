Development Roadmap

Building the future of personal genetic archaeology

Current Status

Phase 1: Research Preview

Hidden Lineage is currently in active research preview mode with full-genome ADMIXTURE runs underway. K=21–30 are in progress, and a balanced global reference panel is under construction. This foundational phase validates the methodology and confirms the ability to trace deep ancestry through ancient DNA.

Development Timeline

1
COMPLETED

Phase 1: Research Preview

Establish methodology and demonstrate feasibility with chromosome 22 analysis. Build core infrastructure and validate approach against known populations.
  • ADMIXTURE pipeline development and testing
  • AADR ancient DNA reference integration
  • K=27 model optimization and cross-validation success
  • Interactive dashboard and visualization
  • Comprehensive documentation and transparency
  • Public website and methodology publication
✅ Completed: December 2024

Technical achievements: Integrated 15,000+ ancient genomes from AADR v62, optimized SNP filtering pipeline (MAF > 0.01, LD pruning r² < 0.2), achieved cross-validation error of 0.63218 at K=27.

Validation results: Successfully reproduced known population structure patterns, identified meaningful ancient connections (e.g., I1877 Neolithic farmer match).

2
IN PROGRESS

Phase 2: Full Resolution

Scale to whole-genome analysis and implement reference panel improvements. This phase will dramatically increase accuracy and provide robust confidence estimates.
  • Whole genome analysis (all 22 autosomes)
  • Reference panel rebalancing and rare population integration
  • Improved K optimization (targeting K=30-40)
  • Bootstrap confidence intervals and uncertainty quantification
  • Enhanced visualization with geographic mapping
  • Comparative validation against academic studies
🔄 Target: January-February 2025

Computational scaling: Moving from ~7K to ~600K SNPs requires 100x more computation. Implementing distributed processing and memory optimization for feasible runtimes.

Reference improvements: Adding undersampled populations (Sub-Saharan Africa, Oceania, Americas), balancing temporal coverage, integrating recent high-coverage ancient genomes.

Statistical enhancements: Block bootstrap for confidence intervals, cross-validation across different K ranges, component stability analysis.

3
PLANNED

Phase 3: Public Platform

Launch user upload capabilities and build interactive platform for public access. Enable users to analyze their own DNA with the same research-grade methods.
  • Secure DNA file upload system (23andMe, AncestryDNA, etc.)
  • Automated analysis pipeline with queue management
  • Personalized interactive results dashboard
  • Comparison tools against reference populations
  • Privacy-first architecture with local processing options
  • Community features and result sharing (opt-in)
🎯 Target: March-May 2025

Privacy architecture: End-to-end encryption, automatic data deletion, local processing options, HIPAA-compliant infrastructure design.

Scalability: Cloud-based processing pipeline with auto-scaling, estimated capacity for 100+ concurrent analyses.

Quality control: Automated contamination detection, coverage assessment, result validation against population structure.

4+
FUTURE

Advanced Features

Explore cutting-edge applications including archaic ancestry, trait analysis, and integration with other genomic methods.
  • Neanderthal and Denisovan ancestry analysis
  • IBD segment analysis for recent connections
  • Trait-associated variant interpretation
  • Integration with Y-chromosome and mitochondrial analysis
  • Machine learning models for ancestry prediction
  • Academic collaboration and peer review publication
🔮 Target: 2025-2026

Archaic ancestry: Implementation of qpAdm and ADMIXTOOLS for formal introgression testing, specialized handling of archaic reference genomes.

IBD analysis: Integration with genealogical-scale analysis using tools like IBDseq, bridging deep and recent ancestry.

Research integration: Collaboration with academic ancient DNA labs, contribution to open science initiatives, peer-reviewed publication pipeline.
